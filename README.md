# MAM VERIFIED CONTACT DETAILS

A premium, production-ready SaaS email verification platform built with Next.js 15.

## Features
- ✅ **Single Email Verification**: Instant SMTP/MX/DNS checks.
- 📂 **Bulk Verification**: Upload CSV/XLSX/TXT for large-scale cleaning.
- 📊 **Interactive Dashboard**: Real-time analytics and statistics.
- 🛡️ **Admin Panel**: Manage users and monitor system health.
- 🎨 **Premium UI**: Dark mode, glassmorphism, and smooth animations.

## Tech Stack
- **Frontend**: Next.js 15, React, TypeScript, Tailwind CSS, Framer Motion.
- **Backend**: Next.js API Routes, Node.js DNS/Net modules.
- **Database**: PostgreSQL (Supabase) with Prisma ORM.
- **Auth**: NextAuth.js (Google & Email).

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Set up Environment Variables**:
   Create a `.env` file based on the `deployment_guide.md`.

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Access the application**:
   - Landing Page: `http://localhost:3000`
   - Dashboard: `http://localhost:3000/dashboard`
   - Admin Panel: `http://localhost:3000/admin`
